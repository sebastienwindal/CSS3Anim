CSS Transitions

  • Two styles
  • A Transition
  • A trigger

Without transitions: HELLO

            
/*  style 1 */
#myText {
  background-color: red;
}

/* style 2 */
#myText:hover {
  background-color: blue;
}
        
            
HELLO

With transitions: GOODBYE

            
/*  style 1 */
#myAnimatedText {
  background-color: red;
  -webkit-transition-property: background-color;
  -webkit-transition-duration: 1.0s;
}

/*  style 2 */
#myAnimatedText:hover {
  background-color: blue;
}
        

GOODBYE

What can you transition?

  • Text properties (color, font-size, etc...)
  • Box properties (width, margin, padding, borders, etc...)
  • Transforms
  • SVG

transition-property

  • transition-property: prop1, prop2;
  • transition-property: all;

transition-duration transition-delay

  • transition-duration: 0.5s;
  • transition-duration: 500ms;
  • transition-delay: 0.5s;
  • transition-delay: 500ms;

transition-timing-function

  • ease-in-out;
  • ease-out;
  • ease-in;
  • linear;
  • cubic-bezier(...);

short-hand notation

.myClass {
    transition-duration: 1s;
    transition-delay: 0.5s;
    transition-property: all;
    transition-timing-function: ease-in-out;
}

.myClass 2 {
    transition-property: color, margin-top;
    transition-duration: 1s, 2.0s;
}

===

.myClass {
    transition: all 1s ease-in-out .5s;
}

.myClass2 {
    transition: color 1s,
                margin-top 1s;
}

Limitations

  • No repeat
  • No reverse
  • No pause
  • No intermediary steps
  • Requires explicit trigger (CSS or javascript)

CSS Animations

Similar to transitions with more features.

  • Start
  • Stop
  • Trigger
  • Intermediary steps
  • auto start, repeats, etc...

Example

Animation automatically starts:

.bouncyDiv  {
    -webkit-animation: bounce-animation 2.0s;
    -webkit-animation-delay: 0.5s;
}

@-webkit-keyframes bounce-animation {
    from  { -webkit-transform: translateY(   0px);   }
    10%   { -webkit-transform: translateY( 200px);   }
    30%   { -webkit-transform: translateY(  66px);   }
    50%   { -webkit-transform: translateY( 200px);   }
    70%   { -webkit-transform: translateY( 160px);   }
    to    { -webkit-transform: translateY( 200px);   } 
}

CSS trigger animation

#shakeIt:hover {
    -webkit-animation: shake-it-animation 0.1s;
    -webkit-animation-iteration-count: infinite;
}

@-webkit-keyframes shake-it-animation {
    from  { -webkit-transform: rotate(  0deg);  }
    25%   { -webkit-transform: rotate(-15deg);  }
    50%   { -webkit-transform: rotate(  0deg);  }
    75%   { -webkit-transform: rotate( 15deg);  }
    to    { -webkit-transform: rotate(  0deg);  }
}

animation-fill-mode

Use forward to prevent animation from rolling back.

.fill-mode-none {
    -webkit-animation: font-size-animation 3.0s ease-in-out;
}
.fill-mode-forwards {
    -webkit-animation: font-size-animation 3.0s ease-in-out forwards;
}
@-webkit-keyframes font-size-animation {
    from    { font-size: 20px; letter-spacing: 1.0; }
    to      { font-size: 60px; letter-spacing: -25px;}
}

pausing animations

  • animation-state: paused|running;
  • Triggered using pseudo-classes
#pausableAnimatedDiv {
    width: 100px;
    height: 70px;
    background-color: yellow;
    border: 2px solid black;
    -webkit-animation: rotate-animation 3s linear infinite;
}

#pausableAnimatedDiv:hover {
    -webkit-animation-play-state: paused;
}
